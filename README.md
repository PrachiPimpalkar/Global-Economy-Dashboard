# Global Economy Dashboard 2010-2025
Power BI Dashboard analyzing World Bank data across 217 countries

## 📊 Project Overview
Interactive Power BI dashboard analyzing 
global economic trends across 217 countries 
using World Bank data (2010-2025)

## 🛠️ Tools Used
- Microsoft Excel (Data Cleaning)
- Power BI (Dashboard & Visualization)

## 📁 Dataset
- Source: World Bank
- Rows: 3,467
- Countries: 217
- Years: 2010-2025

## 💡 Key Insights
- USA = World's Largest Economy (#1)
- India = 7th Largest Economy
- China = Highest GDP Growth
- 2022 = Peak Global Inflation Year
- South Asia = Fastest Growing Region
- MENA = Highest Unemployment Rate
